This small project is an attempt to visualize something many of us have felt or experienced - the ways in which a book might make us feel. To do this, I borrowed some recent advancements in the computer science world of Natural Language Processing. Facebook's large language model BART lies at the core of this project for two primary tasks. The first is to classify which emotion is likely represented in each chunk of text, and the next is to summarize some random paragraphs to give the reader a way to interact with the psychology of the model. Details on the implementation and the full code is publicly available for your convenience at this github repository
